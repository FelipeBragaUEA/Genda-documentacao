Introdução

Este documento especifica os requisitos da “Genda”, software desenvolvido no curso de MPS, fornecendo as informações necessárias para entendimento do projeto e a sua implementação. Genda é um aplicativo de produtividade e organização de tempo, utilizando alarmes e calendários para relembrar datas e horas importantes pro usuário.

Visão

Genda é um aplicativo modelado para obtenção de nota na disciplina de MPS no período 2020-2, na Universidade Estadual do Amazonas - UEA, campus EST - Escola Superior de Tecnologia. O projeto funciona como um aplicativo de produtividade, selecionando hora, data e descrição de um evento, com finalidade de compartilhá-lo com múltiplas pessoas.

    • Regras de negócio

    RN001 - Disponibilidade
    • Descrição: Descrição: O serviço deve permanecer funcionando e disponível por todos os dias, a quaisquer horários.
    • RN002 - Implementação
    • Descrição: O serviço será modelado com a seguinte arquitetura disposta no próximo documento.
    • RN003 - Compatibilidade
    • Descrição: O serviço deve ser compatível com aparelhos móveis Android, versão 11
    • RN004 - Usabilidade
    • Descrição: A instrução ao usuário ocorrerá pela página onde o mesmo pode realizar o download.
    • RN005 - Segurança
    • Descrição: Nenhum dado do usuário deverá ser disposto publicamente, ou disponibilizado para fontes externas, mesmo sigilosas.

    • Requisitos Funcionais

    Lista de Requisitos Funcionais

    RN001 - Fazer Login
    • Descrição: Responsável por logar o usuário no sistema
    • RN002 - Cadastrar usuário
    • Descrição: Responsável por verificar e executar o cadastro do usuário
    • RN003 - Cadastrar Evento
    • Descrição: Responsável por determinar o período de início e fim de um evento, conforme demanda
    • RN004 - Editar Evento
    • Descrição: Alterar informações de um evento
    • RN005 - Listar Evento
    • Descrição: Listar eventos previamente cadastrados
    • RN006 - Alterar participantes no Evento
    • Descrição: Alterar participates do evento, após a criação
    • RN007 - Alterar senha
    • Descrição: Alterar a senha do usuário
    • RN008 - Exibir perfil do usuário
    • Descrição: Exibir os dados do cadastro do usuário na tela
